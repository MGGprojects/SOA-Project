This frontend talks only to the API gateway. The gateway routes requests to the existing microservices for events and calendar exports.

Gateway Health

Architecture Flow

  1. Frontend requests data from the gateway.
  2. Gateway routes `/api/events` to `event-service`.
  3. Gateway routes `/api/exports` to `calendar-export-service`.
  4. User exports the event to Google, Outlook, or ICS.
